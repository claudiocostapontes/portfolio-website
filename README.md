# Claudionor Pontes - Portfólio Profissional

Portfólio pessoal focado em apresentar minha trajetória na área de TI, destacando minha atuação como Arquiteto de Software Sênior, Engenheiro de Sistemas e Desenvolvedor Full Stack.

O design segue uma linha "Dark Tech", otimizado para performance e responsividade, refletindo a seriedade e o rigor técnico exigidos em arquiteturas complexas, microsserviços e segurança da informação.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando um ecossistema front-end moderno e enxuto:

* **Vite**: Ferramenta de build de altíssimo desempenho.
* **Tailwind CSS**: Framework CSS utilitário para estilização ágil e padronizada.
* **HTML5 / CSS3**: Estruturação semântica e customizações visuais nativas.
